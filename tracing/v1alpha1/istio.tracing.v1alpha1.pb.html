Provides configuration parameters for Istio tracing.

Enumerations

TraceSampling.Mode

Controls the mode of sampling.

Name Description
NONE

Do NOT generate trace spans.
MINIMAL

Use the pre-configured minimal trace span generation settings. This will generate trace spans for 1% of all requests (and 100% for client-directed forced traces).
CUSTOM

Use a custom sampling configuration.
ALL

Generate trace spans for 100% of requests, client-requested or otherwise.

Messages

Percent

Identifies a percentage.

Field Type Description
value double

Valid range: [0.0, 100.0]

TraceCollector

TraceCollector holds the configuration for a specific tracing backend.

Field Type Description
zipkinCollector ZipkinCollector (oneof)

Configuration information for a Zipkin trace collector.

TracingRule

TracingRule controls Istio mesh trace collection configuration parameters. It includes the configuration for the tracing backends as well as the rates at which distributed tracing data should be generated.

Example configuration:

apiVersion: tracing.istio.io/v1alpha1
kind: TracingConfiguration
metadata:
  name: meshwide
spec:
  traceCollector:
    zipkinCollector:
      address: zipkin:9411

WARNING: This is not currently supported by Istio.

Field Type Description
serviceName string

Provides a way to tailor the configuration for a specific service. If not specifed, this applies to all services in the namespace of the configured resource. When the namespace is istio-system, the settings will be applied to the entire mesh.

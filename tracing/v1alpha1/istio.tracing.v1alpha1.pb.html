Provides configuration parameters for Istio tracing.

Percent

Identifies a percentage.

Field Type Description
value double

Valid range: [0.0, 100.0]

TraceCollector

TraceCollector holds the configuration for a specific tracing backend.

Field Type Description
zipkinCollector ZipkinCollector (oneof)

Configuration information for a Zipkin trace collector.

TracingConfiguration

TracingConfiguration controls Istio-wide trace collection configuration parameters. It includes the configuration for the tracing backends as well as the rates at which distributed tracing data should be generated.

Example configuration:

apiVersion: tracing.istio.io/v1alpha1
kind: TracingConfiguration
metadata:
  name: meshwide
spec:
  traceCollector:
    zipkinCollector:
      address: zipkin:9411

WARNING: This is not currently supported by Istio.

Field Type Description
traceCollector TraceCollector

Provides configuration for a tracing backend

ZipkinCollector

Configuration for a Zipkin collector.

Field Type Description
address string

Required. Address of the Zipkin service (e.g. zipkin:9411).
endpoint string

Optional. API endpoint of the Zipkin service.

Default: “/api/v1/spans”
